# Project Title: Spendo: An Expense Tracker

## Project Description: This project is aimed to help people track their expenses

## How to run the program 
1. Make sure you have Python and Flask installed.
2. If you dont have flask paste this in your command prompt: "pip install flask"
3. Download the folder "CSProj" and place it on your desktop
4. Open a terminal or command prompt and input "cd Desktop\CSProj"
5. Run the program by inputting "python Spendo.py"
6. Copy the link provided after and paste it into your desired browser (preferrably chrome)
7. Follow the on-screen instructions to enter the amount of money, deductions, additions, classifying and viewing your transactions.

## Features
-Ask for the total balance
-Ask if they will Add, deduct, classify, view purchases, or end the program
-Depending on what they pick the program will add, deduct, classify or view transactions
-When classifying a menu will show to help the user classify their expenses
